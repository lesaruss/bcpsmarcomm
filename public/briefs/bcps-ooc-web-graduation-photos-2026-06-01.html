Key Decisions

Graduate Profile Layout: 2 to 3 Columns
Approved transition from 2-column to 3-column layout for graduation photo pages. This reduces scrolling friction for parents and improves visual hierarchy. Layout tested and confirmed working with left navigation sidebar intact.
Image Optimization Approach
Approved use of Canva background removal and image upscaling tools for low-resolution photos. Method: remove background (to clean yearbook-style blue background), upscale if needed, maintain 262x323px dimensions as standard.

Graduation schedule component to be removed after final event date (Wednesday). No historical archive requirement confirmed. Timeline confirms schedule no longer needed after graduation concludes.
Automated Audit Architecture via Claude Cowork
Approved framework for automated BCPS department page audits using Claude Cowork. Sean A. Russell will build proof of concept this week: define audit requirements document, provide department page snapshot, have Claude evaluate each page and score compliance, run weekly.

Automation Audits Initiative

Objective: Replace manual audit process with automated Claude Cowork verification that runs weekly, scales across all department pages, and reduces WCM response latency.

Phase 1: Proof of Concept (This Week)

  • Document audit requirements (same context provided to WCMs)
  • Create snapshot of all department infrastructure pages
  • Run Claude through 1-2 sample department pages with requirements
  • Verify Claude correctly identifies required elements and compliance gaps
  • If solid, move to Phase 2

Phase 2: Weekly Automation (Scaled)

  • Run automated audit on all department pages every single week
  • Similar to Site Improve model but running on internal infrastructure
  • Auto-generate audit score report for each department

Phase 3: Enhanced Features (Iterative)

  • Add ADA accessibility checker to audit scope
  • Auto-email WCM queue when audit finds issues
  • Auto-send 5-day reminder if WCM has not responded
  • Identify enhanced elements used outside standard layout
  • Create video guides showing approved vs. enhanced element variations

Expected Outcome: Weekly automated compliance reports replace manual audits. WCMs get immediate notification and 5-day response window with reminder. Dashboard shows real-time compliance status. Reduces audit labor from manual to monitoring.
