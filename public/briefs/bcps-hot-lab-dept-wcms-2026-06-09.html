Key Decisions

Hot Lab Format Restructured for Summer
Hot Labs will follow a course-style syllabus beginning this summer. Sean A. Russell will publish a 12-session syllabus so WCMs know what is coming. Each session will have optional pre-work videos. Sessions will include an open Q&A segment at the start and a hands-on mini-workshop component. Recordings of sessions will serve as a content library available to all WCMs.
Video Playlist Autoplay - No Short-Term Fix Available
Angela's attendance page uses a CMS gallery widget, which does not trigger video autoplay. The newsroom page works because it uses an embedded EduVision player - a different technology. Providing EduVision accounts to individual departments is not feasible at scale due to account limits and governance concerns. Sean will note this as a long-term item to explore. For now, departments are limited to manual-play video galleries.
WCM Backup Access Process Clarified
Michelle from Rural Schools Police submitted a form to add a backup WCM (Jessica, identified as Director-level). Sean A. Russell confirmed the form was received. To complete the process, the director must also fill out the Department WCM Roster form so there is an official record. Sean will provision access once the director form is submitted. Sean recommended WCMs bookmark the roster form link for future backup or coverage needs.
Program Spotlight - "No Post" State Handling
Angela asked how to hide a Program Spotlight section when there is no active spotlight content. The recommended approach: copy the postboard widget to a personal resources page (unpublished), then delete the widget from the live page. The resources page serves as a personal WCM toolkit for reusable widgets. This practice will be added to the WCM best practices guide.
