What this is: Find It Fast is the search and category widget on the Back to School page that links out to enrollment forms, calendars, and family resources. It used to be hardcoded, so any update needed a code push from Sean or Claude. It now has its own editor inside the BCPS Web Team Portal, so Felicia can add, edit, reorder, or remove items herself, with no code and no waiting.

Goal

Felicia can update Find It Fast content on her own, and changes appear on the live Back to School page immediately.

Getting In

1
Go to bcpsmarcomm.com/bcps/login
Sign in with your existing BCPS Web Team Portal account, the same one you already use for the Web Team Assignments tracker and other district tools. No new account or password needed.
2
Find "Find It Fast" in the left sidebar
It is listed under the MarComm section, alongside Newsroom and Graphics & Printing. You have been granted access to this page specifically, so it will appear for you even though it will not appear for every Web Team member.
3
You are in the editor
You will see every category and every resource link currently live on the Back to School page, grouped by category.

What You Can Do Here

Edit an existing link or category
Type directly into the Title, URL, or Blurb fields for a link, or the Label for a category. Click Save next to that row when you are done. Save only lights up once you have changed something, so you cannot accidentally save nothing.
Add a new link
Scroll to the bottom of the category the link belongs in and click + Add link. Fill in Title, URL, and a short Blurb, then click Add.
Add a new category
Use the New category box at the bottom of the page. Give it a short slug (a web-safe short name, lowercase with dashes), a Label people will see, and pick an icon from the dropdown.
Reorder items
Use the up and down arrows next to any category or link to change the order it appears in on the public page.
Delete a link or category
Click Delete and confirm. Deleting a category also removes every link inside it, so move links to a different category first if you only meant to retire the category name.
Heads Up
Changes are live immediately. There is no draft or preview step and no undo button. Whatever you save shows up on browardschools.com right away. If you are unsure about a change, ask Sean before deleting something rather than after.

Icon Key Options

Each category has an icon shown next to it on the public page. Pick the one that best matches the category's topic:

Questions or Issues

Something looks wrong on the live page
Double check the row you last edited was saved (the Save button should be greyed out again once it goes through). If it still looks off after a refresh, contact Sean Russell.
You don't see Find It Fast in your sidebar
Your access may not have come through yet, or you may be logged into a different account than the one Sean granted access to. Contact Sean Russell to confirm.
Someone else on the Web Team needs access too
Access to this page is granted per person, the same way document sharing works elsewhere in the portal. Ask Sean Russell to add anyone else who needs it.
