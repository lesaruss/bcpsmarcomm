# K-12 Unlocked -- Broward County Public Schools

## Setup

```bash
cd k12-unlocked
npm install
npm run dev
```

Then open **http://localhost:3000** in your browser.

## What's inside

| Page | Route |
|------|-------|
| Mission Control (Dashboard) | `/` sidebar |
| Meeting Notes (with @mentions) | sidebar |
| My Profile (4 tabs) | sidebar |
| Departments Directory + Profile | sidebar |
| Analytics (quiz to dashboard) | sidebar |
| SuperAdmin Panel | sidebar |
| MarComm / Minutes / WCM Consoles | sidebar |

## Notes
- Node.js 18+ required
- Runs on port 3000 by default (`npm run dev -- --port 3001` to change)
- All data is mocked -- no database required
